# CodeAlpha Language Translator

## Project Overview

This is an AI-powered Language Translation Tool developed as part of the CodeAlpha Artificial Intelligence Internship.

The application allows users to translate text from one language to another using an easy-to-use interface.

## Features

* Translate text into multiple languages
* Automatic language detection
* Simple and user-friendly interface
* Fast and accurate translation

## Technologies Used

* Python
* Streamlit
* Deep Translator

## How to Run

1. Install the required libraries:

   ```
   pip install streamlit deep-translator
   ```

2. Run the application:

   ```
   streamlit run app.py
   ```
